Vamos usar Python com Flask para criar um servidor web simples. 
Certifique-se de ter o Python instalado no seu sistema.

Instalação das Dependências:
Abra o terminal e instale o Flask usando o seguinte comando: pip install Flask

Execução do Aplicativo:
Execute o aplicativo digitando python app.py no terminal. 
Abra seu navegador e acesse http://localhost:5000. 
Você verá a página da aplicação e poderá gerar paletas de cores clicando no botão "Generate Palette".
